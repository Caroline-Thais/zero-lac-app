<h3>Tecnologias utilizadas:</h3>

- Swift
- XCode
- SwiftUI

<h3>Habilidade desenvolvidas:</h3>

- Desenvolvimento de telas de apresentação do app (onboarding screen)
- Criação de views com SwiftUI
- Recuperalção de dados locais usando Structs
- Display de diferentes views usando loop ForEach 
- Utilizacao de scroll view, horizontal view e vertical view
